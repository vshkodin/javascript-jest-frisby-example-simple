### Simple example of API Testing Framework using Javascript, Jest, and Frisby

#### Clone project:
```
git clone https://github.com/vsshk/javascript-jest-frisby-example-simple.git
cd javascript-jest-frisby-example-simple
```
#### Install using npm
```
npm install 
```
#### Run
```
npm run test
```
